## Instructions to build and run the image

Step 1: We need to create a file named as Dockerfile

Step 2: Provide instructions in the Dockerfile

Step 3: We need to build the dockerfile in order to create image
	
	command: docker build . or docker build the path of the file 
	
	
step 4: We need to run the image to create container

	
Finally we get the message like successfully build with the image id.

Commands used

docker build .

docker run image-id
